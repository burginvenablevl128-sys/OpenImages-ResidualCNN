# 课程论文要求覆盖说明

本项目按《人工智能》课程论文要求中的算法复现方向组织，目标是完整呈现数据采集、模型复现、实验验证和结果讨论过程。

论文结构覆盖如下内容：

| 课程要求 | 本项目对应部分 |
|---|---|
| 课程论文来源说明 | 论文第 1 部分，说明课堂 PPT、Open Images、论文、官方文档、代码与大语言模型辅助情况 |
| 摘要 | 论文摘要部分 |
| 引言 | 论文第 3 部分 |
| 算法简介 | 论文第 4 部分，介绍 MLP、CNN、残差连接、损失函数 |
| 数据介绍 | 论文第 5.2 部分，介绍 Open Images V7 与本项目子集 |
| 数据预处理 | 论文第 5.4 部分，说明 resize、标准化、划分、失败日志 |
| 数据可视化 | 数据画像、样本展示、类别分布等图片 |
| 实验环境 | 论文第 5.5 部分 |
| 参数设置 | 论文第 5.5 部分表格 |
| 对照实验 | MLP 与 CNN/Residual CNN 对比 |
| 定量评估 | Accuracy、Macro-F1、混淆矩阵 |
| 定性评估 | 样本图、错误案例与类别混淆分析 |
| 代码运行截图 | 论文中插入 `figure_code_run_result.png` |
| 讨论 | 论文第 7.4 部分 |
| 结论 | 论文第 8 部分 |
| 参考文献 | 论文文末 |
| GitHub/README 加分项 | 项目包含完整 README 与项目结构说明，可上传 GitHub |
| 自构数据集加分项 | 使用 Open Images 官方元数据筛选并下载整理子集 |
| 格式排版加分项 | Word 论文包含规范图题、表题、公式、来源标注和统一排版 |
