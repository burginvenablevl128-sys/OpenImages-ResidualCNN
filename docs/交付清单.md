## 2. 数据与爬虫

- `src/openimages_crawler.py`
  - Open Images 官方元数据下载、类别筛选、图像下载、失败记录与样本表生成。
- `scripts/crawl_openimages.py`
  - 单独运行爬虫的脚本入口。
- `data/raw/`
  - Open Images V7 官方 CSV 元数据。
- `data/processed/openimages_pilot_images.csv`
  - 本文 Pilot 图像样本索引，保留类别、本地路径、URL、作者和许可协议。
- `logs/download_failures.csv`
  - 下载失败样本记录。

## 3. 模型复现代码

- `src/numpy_experiment.py`
  - 无 PyTorch 环境下可直接运行的 MLP 与 CNN-local features 快速实验。
- `src/torch_models.py`
  - MLPBaseline 与 ResidualCNN 的 PyTorch 模型定义。
- `src/train_torch.py`
  - 完整 Residual CNN 训练与测试入口。
- `main.py`
  - 一键运行入口，支持爬虫、实验、绘图和论文生成。

## 4. 实验结果与可视化

- `results/metrics.json`
  - 数据划分、模型指标、混淆矩阵和训练日志的结构化结果。
- `results/model_comparison.csv`
  - 模型性能对比表。
- `results/training_log.csv`
  - 每轮训练损失、训练准确率和验证准确率。
- `figures/`
  - 论文中使用的框架图、数据看板、样本图、模型结构图、指标图、训练曲线、混淆矩阵和运行结果图。

## 5. 复现命令

```powershell
python -m pip install -r requirements.txt
python main.py --all --per-class 8
```

只生成论文：

```powershell
python main.py --report
```

完整 PyTorch Residual CNN 训练：

```powershell
pip install torch
python -m src.train_torch
```
